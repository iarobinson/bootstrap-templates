User Card

This card can be used to display a list of users with their avatar and buttons to interact with them. Works well on many screen sizes (mobile, desktop & tablet).

This would be where you would add some truncated text to link to a blog. This blob is ~150 characters so you can use this as a way to understand the space for the copy.
